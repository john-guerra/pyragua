Pyragua 
-------
http://quimbaya.utp.edu.co/projects/pyragua


Qué es Pyragua?
---------------
PYRAGUA es un entorno de desarrollo para la manipulación de código en lenguaje Python 
diseñado por los integrantes del semillero de investigación de la Universidad Tecnológica de Pereira PYROX.

Entre las principales caracterésticas actuales del Pyragua podemos destacar:

    >> Coloreado de texto.
    >> Comentado y descomentado de bloques de código.
    >> Identado y desidentado de bloques de texto.
    >> Detección y configuración de los fines de línea.
    >> Autocompletado.
    >> Pánel pyShell
    >> La posibilidad de abrir varios archivos .py al mismo tiempo.
    >> Ejecución de las aplicaciones que se realicen en él.

Instalación
-----------
Después de obtener Pyragua se descomprime
    promt$ unzip pyragua_xxxx.zip
    promt$ cd pyragua_xxxx
puede ejecutarlo de inmediato bajo windows o linux
    promt$ python pyragua.py
También lo puede instalar en linux
    promt# python setup.py install (como root)
    promt$ pyragua.py

Interpretación para el manejo de versiones
------------------------------------------
Cada versión será caracterizada por tener en su nombre una parte conformada por
3 carácteres numéricos separados por el carácter punto. El significado es el siguiente:
Nombre #a.#b.#c --> #a: Número de la versión estable
                    #b: Número de la versión en desarrollo
                    #c: Número de la revisión
